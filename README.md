# 永恒之塔2 制作成本计算器生成工具

## 介绍 📜

这是一个基于 Python 的命令行工具，其核心功能是：**将 CSV 格式的物料价格数据和产品配方（BOM）转换为一个独立的、功能齐全的 HTML 网页**。

生成的 HTML 页面——**制作成本计算器**——支持用户在浏览器中直接进行实时、交互式的成本核算。

主要功能亮点：
* ✅ **生成交互式网页：** 脚本自动嵌入所有数据和 JavaScript 逻辑，生成一个独立的 `index_generated.html` 文件，无需服务器即可运行。
* ✅ **种族本地化命名：** 根据用户选择（天族/魔族）自动切换物料和产品的显示名称。
* ✅ **实时价格调整：** 用户可在 HTML 页面中直接修改物料交易行价格，成本实时更新。
* ✅ **配方树 (BOM) 展示：** 递归显示产品的全部制作结构和物料用量。
* ✅ **成功率模拟：** 支持自定义成功率，并自动计算所需的材料倍数。
* ✅ **职业筛选：** 方便用户查看特定职业相关的物料。
* ✅ **数据导出：** 一键导出详细的成本分析报告。

## 运行环境要求 💻

本项目需要 Python 3 环境及以下依赖库：

* **Python:** 3.6 或更高版本
* **pandas:** 用于处理 CSV 文件
* **chardet:** 用于自动检测 CSV 文件编码

首次运行脚本时，它会执行依赖自检，若发现缺失，将提示并尝试使用 `pip` 自动安装。
